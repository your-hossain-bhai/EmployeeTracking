This directory contains assets for the Smart Employee app.

## Structure
- images/ - App images and logos
- icons/ - Custom icons
- fonts/ - Custom fonts (Roboto is referenced in pubspec.yaml)

## Adding Assets
1. Add asset files to the appropriate subdirectory
2. Reference them in pubspec.yaml under flutter: assets:
3. Use AssetImage() or Image.asset() to load them
